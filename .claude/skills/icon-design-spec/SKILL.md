# 图标设计说明文档

## 整体规格
- **画布尺寸**: 512×512 像素
- **背景**: 白色圆角正方形
- **风格**: 简洁现代，金属质感

---

## 详细构成元素

### 1. 背景层
| 属性 | 值 |
|------|------|
| 形状 | 圆角正方形 |
| 位置 | 距离边缘各 24px |
| 尺寸 | 464×464 px |
| 圆角半径 | 80px |
| 填充色 | 纯白 `#FFFFFF` |
| 边框 | 浅灰 `#E5E7EB`，2px |

### 2. 中心字母 Z
| 属性 | 值 |
|------|------|
| 位置 | 画布正中心 (256, 256) |
| 倾斜角度 | **向右 20 度** |
| 字体风格 | 粗体无衬线，几何感强 |

**Z 字母路径坐标**（倾斜前）:
```
起点: (156, 180)
→ 右上: (356, 180)   // 顶部横杠
→ 斜下: (200, 340)   // 中间斜杠终点
→ 右下: (356, 340)   // 底部横杠起点
→ 终点: (156, 372)   // 左下角
```

**Z 字母尺寸**:
- 总宽度: 200px
- 总高度: 192px
- 笔画宽度: 约 40px

### 3. 金色渐变 (应用于 Z 字母)
使用**线性渐变**，方向：左上 → 右下

| 位置 | 颜色 | 色值 |
|------|------|------|
| 0% | 亮金色 | `#FBBF24` |
| 50% | 中金色 | `#F59E0B` |
| 100% | 深金色 | `#D97706` |

### 4. 阴影效果 (仅 Z 字母)
| 属性 | 值 |
|------|------|
| 类型 | 投影 (Drop Shadow) |
| 水平偏移 | 0px |
| 垂直偏移 | 4px |
| 模糊半径 | 8px |
| 颜色 | 深棕色 `#92400E` |
| 透明度 | 30% |

### 5. 装饰边框
| 属性 | 值 |
|------|------|
| 形状 | 与背景圆角正方形重合 |
| 位置 | 距离边缘各 24px |
| 圆角半径 | 80px |
| 边框色 | 金色 `#FBBF24` |
| 边框宽度 | 3px |
| 透明度 | 30% |
| 填充 | 无 |

## 绘制步骤（供设计师参考）

1. **创建画布** - 512×512px
2. **绘制背景** - 白色圆角矩形，距边缘 24px，圆角 80px
3. **添加背景边框** - 浅灰 2px 描边
4. **绘制 Z 字母** - 使用上述路径坐标，笔画粗壮有力
5. **应用金色渐变** - 左上到右下的三色渐变
6. **添加阴影** - 向下 4px 投影，增加立体感
7. **旋转 Z 字母** - 整体向右倾斜 20 度
8. **添加装饰边框** - 金色 3px，30% 透明度，与背景同尺寸

---

## 关键特征（用于风格一致性）
- 字母 **占据中心主导位置**
- 金色渐变赋予**高级感/金属质感**
- 20 度倾斜增加**动感和现代感**
- 白色背景 + 圆角保持**简洁友好**
- 轻微阴影增加**层次深度**
