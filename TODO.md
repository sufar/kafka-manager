### 使用说明

1. **运行数据库迁移**
```bash
# 在服务器重启时会自动运行迁移
# 或者手动复制迁移文件到正确位置
cp migrations/006_create_consumer_group_lag_history.sql <sqlite-migration-path>
```

2. **安装前端依赖**
```bash
cd ui
npm install
```

3. **启动服务**
```bash
# 后端
cargo run

# 前端
cd ui
npm run dev
```

4. **查看 Lag 图表**
- 访问 Consumer Groups 页面
- 点击任意 Consumer Group 名称
- 在详情页查看 Lag 趋势图

### 功能特性

- **自动刷新**: 默认每 5 秒刷新一次，可选择 3s/5s/10s/30s
- **暂停/继续**: 可随时暂停自动刷新
- **历史数据**: 显示最近 1 小时的 Lag 趋势
- **数据持久化**: 后台每 5 秒采集一次数据，保留 24 小时
- **实时显示**: 最新 Lag 值实时更新，支持颜色告警（绿色<1K<黄色<红色）
- 批量删除 Topic
- 最近访问历史
- UI 细节优化
